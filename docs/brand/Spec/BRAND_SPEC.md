# VetCare — Бранд спецификация и дизайн-система

**Версия 1.0** · Задание за изграждане на уебсайт и продуктов интерфейс

Този документ дефинира визуалната идентичност на VetCare. Той служи като задание към дизайнери и разработчици — съдържа точните цветове, типография, правила за лого и компоненти, готови за пряко прилагане.

---

## 1. За марката

**VetCare** е мениджмънт система / SaaS платформа за ветеринарни клиники. Продуктът помага клиниките да следят графиците на обезпаразитяване, ваксинации и прегледи, и да изпращат навременни напомняния, така че нищо да не се пропуска.

**Същност на марката:** надеждност, грижа, технологична прецизност.

**Тон:** професионален, но топъл. Внушава сигурност (като медицинска институция) и същевременно приветливост (грижа за животните). Не е студен корпоративен, нито прекалено игрив.

**Символ:** Рекс — куче, изобразено в low-poly (полигонен) стил. Полигоните носят асоциация с дигитални технологии; кучето носи топлота и емоционалната причина зад продукта. Седящата поза внушава стабилност и спокойствие.

## 3. Употреба на цветовете (кой цвят за какво)

**Фонове**
- Основен фон на страницата: `#FFFFFF`
- Редуващи се секции: `#F4F8FA` или `#E3F1F3`
- Тъмни секции (хедър/футър/hero): `#0B2F39`

**Текст**
- Заглавия и основен текст върху светло: `#0B2F39`
- Второстепенен текст / описания: `#134D5A`
- Текст върху тъмен фон: `#FFFFFF` (основен) или `#E3F1F3` (по-мек)

**Интерактивни елементи**
- Основни бутони (primary CTA): фон `#207885`, текст `#FFFFFF`
- Бутони при hover: `#134D5A`
- Връзки: `#207885`; при hover — подчертаване + `#134D5A`
- Акценти върху тъмен фон: `#60AEBA`

**Състояния (предложение — допълни при нужда)**
- Успех: `#207885` (в палитрата) или зелен `#2E9E6B`
- Предупреждение: `#E0A23B`
- Грешка: `#D1495B`
- Тези три са извън основната палитра и служат само за функционални сигнали.

## 4. Контраст и достъпност (WCAG)

Всички основни комбинации са проверени по WCAG 2.1. За здравен софтуер достъпността е задължителна.

| Комбинация | Контраст | Ниво |
|------------|----------|------|
| `#0B2F39` върху бяло | 14.2:1 | AAA ✓ |
| `#134D5A` върху бяло | 9.4:1 | AAA ✓ |
| `#207885` върху бяло | 5.1:1 | AA ✓ |
| Бяло върху `#0B2F39` | 14.2:1 | AAA ✓ |
| `#E3F1F3` върху `#0B2F39` | 12.3:1 | AAA ✓ |
| `#60AEBA` върху `#0B2F39` | 5.6:1 | AA ✓ |

**Правила:**
- `#207885` като текст се ползва само на размер ≥ 18px или удебелен (заради 5.1:1).
- За дребен текст върху светло винаги ползвай `#0B2F39` или `#134D5A`.
- Никога не слагай `#207885` текст върху `#E3F1F3` за дребен шрифт (само 4.4:1).
- `#D1495B` (Error) директно върху `#0B2F39` дава само ~3.25:1 — не минава AA за дребен текст.
  За грешка/error текст върху тъмен фон ползвай по-светъл нюанс:
  `color-mix(in srgb, #D1495B 55%, white)` (~RGB 230,155,165) → ~6.9:1 върху `#0B2F39`. Виж
  `--color-error-on-dark` в `tokens.css`.

## 5. Типография

**Шрифт на марката: Poppins** (геометричен sans-serif, Google Fonts, безплатен).

Poppins носи модерен, приветлив и достъпен характер — заоблените геометрични форми пасват на „грижовната технология", която VetCare представя.

**Зареждане (уеб):**
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700&display=swap" rel="stylesheet">
```

### Тегла и роли

| Роля | Тегло | Размер (desktop) | Цвят |
|------|-------|------------------|------|
| H1 / Hero | 700 Bold | 48–64px | `#0B2F39` |
| H2 | 700 Bold | 36–40px | `#0B2F39` |
| H3 | 600 SemiBold | 24–28px | `#0B2F39` |
| Основен текст | 400 Regular | 16–18px | `#134D5A` |
| Под-надписи / етикети | 500 Medium | 13–14px, letter-spacing 1px | `#134D5A` |
| Бутони | 600 SemiBold | 15–16px | според бутона |

**Резервни шрифтове (fallback):** `'Poppins', -apple-system, 'Segoe UI', Roboto, sans-serif`

> **Бележка от имплементацията (2026-07-03):** Poppins няма кирилски subset в Google Fonts
> (само `latin` / `latin-ext` / `devanagari`). Тъй като сайтът е изцяло на български,
> текстовото съдържание (body copy) реално **не се изобразява с Poppins** — тихо пада на
> резервния системен шрифт (`-apple-system` / `Segoe UI` / `Roboto`). Poppins реално се
> вижда само върху латинския текст: wordmark-а „VetCare" и цифрите (цени, брой). Това е
> съзнателно решение, не бъг: по-малко/по-леки файлове с шрифтове (браузърът изобщо не
> тегли subset без съвпадащ текст) — сайтът достига **100/100 Lighthouse Performance**
> с тази конфигурация. Ако кирилицата трябва видимо да се изобразява в марковия шрифт,
> алтернатива е **Montserrat** (потвърдено съдържа `cyrillic`/`cyrillic-ext`) — това би било
> съзнателна промяна на спецификацията, не техническа поправка.
>
> Секцията „История" (6.5) вече не ползва отделен serif шрифт (виж бележка в брифовете) —
> ползва Poppins/fallback като останалата част от сайта, за да се избегне зареждане на
> втори font family.

## 6. Лого

### Основна форма
Логото се състои от: **символ (Рекс)** разположен над **wordmark-а „VetCare"**, с центриран под-надпис отдолу.

- „Vet" се изписва в `#0B2F39` (Deep Petrol)
- „Care" се изписва в `#207885` (Teal)
- Под-надпис („VETERINARY SOFTWARE" или финалния избран текст) — `#134D5A`, главни букви, letter-spacing ~8px
- Шрифт: Poppins Bold за wordmark, Poppins Regular за под-надписа

### Символ
Рекс се ползва в **опростената low-poly версия (~182 полигона)** — еднаква в логото и във favicon-а, за консистентност. Детайлната версия (MED, ~1835 полигона) се пази само за големи декоративни изображения (hero), не за марковия знак.

### Варианти (за изработка)
- **Основно** (тъмен текст) — за светъл фон ✓ готово
- **Светло** (бял/Mist текст) — за тъмен фон *(предстои)*
- **Само символ** — Рекс без текст, за тесни пространства *(налично)*
- **Хоризонтално** — символ вляво, текст вдясно (за тесни хедъри) *(предстои)*

### Правила за употреба
- Минимално свободно пространство около логото: равно на височината на буквата „V".
- Минимална височина: 32px (под това ползвай само символа / favicon).
- Не разтягай, не върти, не променяй цветовете, не разменяй цветовете на „Vet"/„Care".
- Върху снимки логото се поставя само върху достатъчно контрастна зона (или в светлата/тъмната му версия).

## 7. Иконография и илюстрации

- Стилът на иконите следва **линеен (outline) стил**, дебелина на линията 1.5–2px, цвят `#207885` или `#134D5A`.
- Полигонният (low-poly) стил е запазен **само за Рекс** — не се прилага върху други илюстрации, за да остане символът разпознаваем.
- За декоративни графики може да се ползват едри тюркоазени геометрични форми от палитрата (триъгълници, фасети), като намек за символа.

## 8. CSS дизайн-токени (готови за копиране)

```css
:root {
  /* Brand core */
  --color-primary:      #0B2F39;  /* Deep Petrol — текст, заглавия */
  --color-secondary:    #207885;  /* Teal — акценти, бутони, връзки */
  --color-tertiary:     #134D5A;  /* Mid Teal — второстепенен текст */

  /* Extended */
  --color-light-teal:   #60AEBA;  /* акценти на тъмен фон */
  --color-soft-teal:    #4493A0;  /* графики */
  --color-pale-aqua:    #AFD3D8;  /* фонове, разделители */
  --color-mist:         #E3F1F3;  /* светли фонове, карти */

  /* Neutrals */
  --color-white:        #FFFFFF;
  --color-offwhite:     #F4F8FA;

  /* Functional (извън основната палитра) */
  --color-success:      #2E9E6B;
  --color-warning:      #E0A23B;
  --color-error:        #D1495B;
  /* --color-error на --color-primary фон измерва само ~3.25:1 (под AA) — виж раздел 4 */
  --color-error-on-dark: color-mix(in srgb, var(--color-error) 55%, white); /* ~6.9:1 */

  /* Typography */
  --font-brand: 'Poppins', -apple-system, 'Segoe UI', Roboto, sans-serif;
  --weight-regular: 400;
  --weight-medium:  500;
  --weight-semibold:600;
  --weight-bold:    700;

  /* Surfaces */
  --bg-page:    var(--color-white);
  --bg-alt:     var(--color-offwhite);
  --bg-dark:    var(--color-primary);

  /* Text */
  --text-main:  var(--color-primary);
  --text-muted: var(--color-tertiary);
  --text-on-dark: var(--color-white);

  /* Interactive */
  --btn-bg:        var(--color-secondary);
  --btn-bg-hover:  var(--color-tertiary);
  --btn-text:      var(--color-white);
  --link:          var(--color-secondary);
  --link-hover:    var(--color-tertiary);
}
```

### Примерни компоненти

```css
/* Основен бутон */
.btn-primary {
  background: var(--btn-bg);
  color: var(--btn-text);
  font-family: var(--font-brand);
  font-weight: var(--weight-semibold);
  font-size: 16px;
  padding: 12px 28px;
  border: none;
  border-radius: 10px;
  cursor: pointer;
  transition: background .2s ease;
}
.btn-primary:hover { background: var(--btn-bg-hover); }

/* Връзка */
a { color: var(--link); text-decoration: none; }
a:hover { color: var(--link-hover); text-decoration: underline; }

/* Заглавие */
h1 {
  font-family: var(--font-brand);
  font-weight: var(--weight-bold);
  color: var(--text-main);
  font-size: clamp(2rem, 5vw, 4rem);
  letter-spacing: -0.02em;
}
```

## 9. Препоръки за уеб сайта (приложение на системата)

- **Hero секция:** тъмен фон `#0B2F39`, бяло заглавие, един акцентен бутон в `#207885`. Тук може да се ползва детайлният Рекс (MED) като голяма илюстрация.
- **Раздели с функции:** редуващи се фонове бяло / `#F4F8FA`, икони в линеен стил, заглавия `#0B2F39`.
- **CTA блок:** фон в тюркоаз или петрол, контрастен бутон.
- **Футър:** тъмен фон `#0B2F39`, текст `#E3F1F3`, логото в светлата му версия.
- **Заобленост:** използвай умерено заоблени ъгли (8–12px), в духа на заоблените форми на Poppins.
